# Validador de Cédula Dominicana 

Este es un proyecto sencillo realizado en **JavaScript** que valida la estructura de la Cédula de Identidad y Electoral de la República Dominicana utilizando el **Algoritmo de Módulo 10**.

##  Características
- Valida cédulas de 11 dígitos.
- Funciona con o sin guiones.
- Interfaz limpia y responsiva.
- Validación en tiempo real.
